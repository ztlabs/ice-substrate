
## Build & Run

To build the chain, execute the following commands from the project root:

```
$ cargo build --release
```

To execute the chain, run:

```
$ ./target/release/amax-eva --tmp --ws-external --rpc-external --rpc-cors all --dev
```

metamask setting:

```
rpc url    :http://127.0.0.1:9933
chainid    :1998
coin symbol:AMAX
```
